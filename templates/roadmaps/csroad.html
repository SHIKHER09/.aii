1st year

  • Start learning about new programming languages.
  • Try and explore different domains like Machine learning, Cloud computing, AI/ML, Cyber Security etc.
  • Make accounts on the platforms:
    • linkedin
    • leetcode
    • Github
    • Codechef
  • Learn about how to maintain your CGPA.
  • Become consistent in coding everyday
  • Now we have to study the basics of OOPS.
  • Read DSA concepts.
  • Start the basics of Web Development. (HTML, CSS)

2nd year

  • Consistently study the advanced concepts of DSA (tree, graphs, DP … etc)
  • Study Javascript too.
  • Explore different fields like AIML, Cybersecurity, Web/App development, etc, and decide your niche technology in which you’d want to work.
  • Make 1 or 2 small projects with Web Development
  • Give programming contests on different websites like CodeChef, Leetcode, etc.
  • Study some libraries and frameworks of JS and CSS like react JS and tailwind CSS
  • Start with getting the basics of CS concepts like OS, CN, and DBMS.

3rd year

  • Evaluate your profile and build a personal resume.
  • Build a personal portfolio website and upload all your achievements on it.
  • Assess your technical skills and learn about Full-stack development.
  • Learn backend technologies.
  • Learn about system design and Software engineering roles.
  • Start posting all your achievements and build connections and networks on LinkedIn and Twitter.
  • Start or try to find any internship opportunities.
  • Post your ideas and learn from other’s projects.
  • Make a major full-stack project.

4th year

  • Maintain your CGPA.
  • Hone your skills more and develop more projects.
  • Search and apply for more job opportunities and full-time roles.
  • Start freelancing.
  • Develop your soft skills, read aptitude, and learn proper interview etiquette
